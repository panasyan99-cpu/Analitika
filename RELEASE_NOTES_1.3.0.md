# Analitika Web 1.3.0

## Основные изменения

- Склад Baserow отображает обзор, сувениры, касты, проблемные позиции, движение и поставки одной непрерывной страницей.
- Боковая и мобильная навигация склада работает по якорям и не скрывает аналитические блоки.
- Дублирующий переключатель «Раздел склада» удалён из основной области.
- Блок курса VND/USD корректно перестраивается на телефоне: подпись и поле ввода не перекрывают друг друга.
- В Sonu термин «участники групп» заменён на «виды камней» в интерфейсе, таблицах и Excel-выгрузке.
- Усилено скрытие служебных виджетов состояния Streamlit.

## Проверка

- 72 автоматических теста.
- Все четыре режима открыты через Streamlit AppTest без исключений.
- Проверен запуск реального headless-сервера Streamlit и endpoint состояния приложения.
