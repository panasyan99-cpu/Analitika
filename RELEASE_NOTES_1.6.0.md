# Analitika Web 1.6.0 — Metal filters and Sonu AI order report

## Новый основной отчет

Для обычной аналитики и сравнения периодов используется единый файл с иерархией «Магазин → Камень → Проба → Номенклатурная группа».

## Сравнение периодов

- глобальные галочки «Серебро», «Золото и платина», «Другое»;
- фильтр пересчитывает всю страницу и оба периода;
- отдельная диаграмма количества и USD по пробам;
- таблица с детализацией до камня и номенклатурной группы.

## Sonu

В релиз включен полный управленческий отчет Sonu с сетевыми остатками, AI-приоритетом заказа, большой сводкой по ассортименту и двумя самостоятельными типами браслетов.
