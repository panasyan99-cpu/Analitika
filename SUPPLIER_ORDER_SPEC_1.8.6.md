# Заказ поставщику — уточнения 1.8.6

## Полный состав комплекта

Ключ комплекта: `режим заказа + нормализованный камень + исходное название Set#`.
Категория определяется максимальными продажами одного изделия, но после классификации карточка содержит все изделия этого ключа. Фильтры категории не применяются повторно к отдельным строкам.

## ТВП

Переключатель ТВП отбирает комплекты:

- включён — комплект содержит хотя бы одну строку `ТВП > 0`;
- выключен — комплект содержит хотя бы одну строку `ТВП ≤ 0`.

Внутри отобранного комплекта показывается полный состав. Отдельного expander-блока ТВП нет.

## Смена источника

Source ID — SHA-256 содержимого Excel. При успешной загрузке нового Source ID приложение удаляет все persisted workspaces другого Source ID. Поля Streamlit изолированы source hash. Идентичный файл считается тем же заказом и может продолжить черновик.

## Аналитические группы камней

`Green Stones`, `Other Topaz`, `Other Stones` используются только как верхнеуровневые фильтры. В сущности OrderItem, карточке, распределении размеров и Excel хранится конкретное нормализованное название. Если конкретный камень не определяется, используется `Камень не распознан`.
