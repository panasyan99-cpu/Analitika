# Analitika Web 1.9.7 — Ring-size SKU Clipboard

## Копирование артикула

В карточках распределения размеров колец рядом с текстовым SKU добавлена компактная кнопка `⧉`.

При нажатии:

- в буфер копируется только артикул;
- начальные и конечные пробелы, табуляции и переносы строк удаляются;
- внутреннее содержимое SKU сохраняется без изменений;
- успешное действие подтверждается символом `✓`.

Используется браузерный Clipboard API. Для окружений, где он недоступен, предусмотрено резервное копирование через временное поле.

## Совместимость

Логика заказа, облачные черновики, рекомендации, Limited Order и распределение размеров не изменялись. Сохранённые заказы Cloudflare R2 совместимы с версией 1.9.7 без миграции.
