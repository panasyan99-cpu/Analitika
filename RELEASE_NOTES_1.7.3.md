# Analitika Web 1.7.3

## Классификация браслетов

В Sonu появился отдельный прозрачный блок классификации браслетов. Он показывает, какие модели определены однозначно, какие классифицированы по модельному семейству, а какие были спорными и распределены по правилу 50/50. Для каждой спорной модели видны продажи, остаток, выбранная группа и объяснение решения.

Полная классификация доступна на странице и в Excel-выгрузке.
