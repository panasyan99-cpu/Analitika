# Analitika Web 1.8.7

## Исправление

При открытии некоторых категорий, особенно блока «Без комплекта», версия 1.8.6 могла завершаться с `NameError`. Причина: `_render_set_card()` обращалась к `parsed.source_hash`, хотя объект `parsed` не передавался в эту функцию.

В 1.8.7:

- `source_hash` передаётся явно по цепочке `_render_sets_group → _render_set_card → _render_item_row`;
- поля количества остаются изолированными по исходному Excel;
- добавлен регрессионный тест на реальную отрисовку карточки комплекта;
- бизнес-логика заказа не изменялась.
