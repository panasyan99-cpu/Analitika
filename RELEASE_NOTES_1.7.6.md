# Analitika Web 1.7.6

## Каталог браслетов Sonu

Файл `sonunew.xlsx` использован как большой базовый каталог текущего ассортимента. По фотографиям предварительно классифицированы 340 из 350 SKU и 209 из 218 модельных семей.

Для проверки пользователем оставлены только 10 SKU из 9 пограничных модельных семей. Кнопка «Разобрать модели» открывает крупное фото представителя семьи и два варианта: «С затяжкой / центральная композиция» или «Без затяжки / полный круг». Одно решение сразу применяется ко всем SKU семьи.

Скрытое распределение 50/50 удалено. До ручного решения пограничные модели явно помечены как требующие классификации и не попадают в два основных браслетных блока.
