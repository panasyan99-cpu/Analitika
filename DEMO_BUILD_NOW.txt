КАК ПОЛУЧИТЬ УСТАНОВЩИК ДЛЯ ДЕМО

1. Скопируйте содержимое этой папки в локальный репозиторий Analitika.
2. Папку .git не удаляйте и не заменяйте.
3. GitHub Desktop: Commit to main -> Push origin.
4. GitHub: Actions -> Build Windows installer -> Run workflow.
5. После Success внизу появится artifact Analitika_Setup_1.1.5_DEMO.
6. Скачайте artifact, распакуйте ZIP и запустите Analitika_Setup_1.1.5.exe.
