# Спецификация рекомендаций заказа поставщику 1.9.5

## 1. Последовательность расчёта

1. Определить базовую рекомендацию по типу изделия, фактическим продажам, TT, общему остатку, внутреннему остатку для рекомендации, ТВП и балансу настоящего комплекта.
2. Для режима «Камни» определить шкалу камня.
3. Для остальных цветных камней уменьшить базовое количество на 1.
4. Применить финальное правило партии: 1 → 0, 2 → 3, 3+ без изменения.
5. Не записывать рекомендацию в заказ без кнопки «Согласен с рекомендацией».

## 2. Полная шкала

Полная шкала, совпадающая с жемчугом, действует для:

- Ruby;
- Sapphire любого нормализованного вида;
- Topaz любого нормализованного вида;
- Moissanite;
- Emerald;
- Created Emerald;
- Onyx;
- Black Spinel;
- Green Agate.

В режиме «Жемчуг» полная шкала используется всегда.

## 3. Остальные цветные камни

Для всех остальных камней итог базового движка уменьшается на 1 до применения минимальной партии.

Примеры:

- 5 серёг → 4;
- 4 кольца → 3;
- 3 кольца → 2 → финально 3;
- 2 подвески → 1 → автоматической рекомендации нет;
- 10 пусетов → 9.

Поправка действует для покрытия спроса, TT, слабых и нулевых комплектов, «Без комплекта» и балансировки по ТВП.

## 4. Минимальная партия

- расчёт 0: не рекомендовать;
- расчёт 1: не рекомендовать;
- расчёт 2: рекомендовать 3;
- расчёт 3+: рекомендовать рассчитанное количество.

Ручное количество и Limited Order не ограничиваются.

## 5. ТВП

- Положительный ТВП конкретного SKU блокирует автоматическую рекомендацию этого же SKU.
- В настоящем комплекте серьги в пути могут рекомендовать кольца, а кольца в пути — серьги.
- В «Без комплекта» перекрёстная балансировка запрещена.

## 6. Остатки

- «Общий остаток» = «Всего» минус единая точка 20 / 20NDC / Princess Hang.
- TT и 63 входят в общий остаток и показываются отдельно.
- Внутренний остаток для рекомендации дополнительно исключает 63.
