# Changelog

## 1.1.0 RC1 — Stable one-page analytics
- Исправлена ошибка `StreamlitDuplicateElementKey` при первой прокрутке и переключении магазинов.
- Убрана повторная отрисовка интерактивного блока внутри страницы магазина.
- Все фильтры получили отдельное пространство ключей Streamlit.
- Поставщики загружаются и отображаются автоматически после загрузки файла.
- Добавлены видимые фирменные разделители между крупными блоками отчета.
- После сводки, магазина и поставщиков выводится отдельный аналитический блок.
- В интерактивной аналитике выводы перестраиваются вместе с выбранными фильтрами.
- Экспорт Excel теперь формируется только по кнопке и не замедляет переключение фильтров.
- Сохранена единая страница: Сводка → Магазины → Интерактивная аналитика → Поставщики → Экспорт.

## 1.0.3 — One Page Report
- Вся аналитика находится на одной большой странице после единственной загрузки файла.
- Левое меню работает как навигация-якорь.
