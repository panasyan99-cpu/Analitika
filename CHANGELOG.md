# Аналитика 1.1.6

- Добавлены каналы обновлений **Stable** и **RC**.
- Добавлены сохраняемые настройки проверки обновлений.
- Обновление скачивается в `Документы/Analitika/Updates`.
- Установщик обновляет существующую копию приложения по постоянному AppId и сохраняет выбранную при первой установке папку.
- Добавлена автоматическая сборка установщика и SHA-256 checksum через GitHub Actions.
- Пользовательские отчеты, настройки и логи хранятся отдельно от приложения.
- При отсутствии интернета приложение продолжает работать без блокирующих ошибок.
