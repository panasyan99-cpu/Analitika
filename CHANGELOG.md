# Changelog

## 1.1.0 — 2026-07-10

- Добавлена поддержка одной общей выгрузки по всем магазинам.
- Возвраты полностью исключены из обработки.
- Добавлены отдельные отчеты по магазину и периоду.
- Добавлено сравнение нескольких периодов одного магазина.
- Добавлены диаграммы на SUMMARY, листах магазинов и COMPARE.
- Обновлены правила Top Stones, Pearls и Other Stones.
- Moissanite имеет абсолютный приоритет.
- Все Sapphire объединяются в Blue Sapphire.
- Onyx и Jasper объединяются по синонимам.
- Добавлена автоматическая сборка установщика через GitHub Actions.
- Добавлена проверка обновлений через GitHub Releases.
- Добавлена подпись: Developed by Vladimir Panasyan.
