# Changelog

## 1.0.3 — One Page Report
- Вся аналитика теперь находится на одной большой странице после единственной загрузки файла.
- Левое меню работает как навигация-якорь: Сводка, Магазины, Интерактивная аналитика, Поставщики, Экспорт.
- Убрано переключение страниц через `session_state`, вызывавшее StreamlitAPIException.
- Поставщики без указанного значения, а также служебное значение «Сеть», объединяются в `Other`.
- Добавлены сравнительные диаграммы долей поставщиков по количеству и выручке.
- Для выбранного поставщика добавлены разрезы по магазинам, сегментам, камням и номенклатурным группам.
- Служебные подразделения не создают ложный магазин «Сеть».
