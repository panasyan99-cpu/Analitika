# Analitika Web Roadmap

## 1.1.0 — Production baseline
- Единая загрузка общей выгрузки
- Сводка по сети
- Аналитика магазинов
- Интерактивный разрез: сегмент → камень → номенклатурная группа
- Аналитика поставщиков
- Динамические выводы
- Условная боковая навигация

## Следующие версии
- Сравнительный анализ периодов и магазинов
- Пробы и категории металлов: Серебро / Золото / Другие
- Аналитика продавцов по магазинам, сегментам, камням и товарным группам
- История отчетов и сохраненные фильтры
- Расширенная AI-аналитика
