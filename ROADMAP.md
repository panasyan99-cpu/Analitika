# Analitika Web — политика обновлений

Публичные планы развития не фиксируются, так как приоритеты могут меняться.

Все фактически выпущенные изменения публикуются:

- в разделе **«О программе»** внутри приложения;
- в файле **CHANGELOG.md**.

Аналитика продавцов исключена из продукта и не входит в будущие итерации без отдельного решения.
