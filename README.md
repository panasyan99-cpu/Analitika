# Analitika Web 1.0.3 — One Page Report

Внутренняя BI-платформа Princess Jewelry.

## Основной сценарий

1. Пользователь один раз загружает общую Excel-выгрузку.
2. На одной странице открываются все блоки отчета:
   - сводка по сети;
   - аналитика магазинов;
   - интерактивный разрез сегмент → камень → номенклатурная группа;
   - аналитика поставщиков;
   - экспорт.
3. Левое меню прокручивает страницу к выбранному блоку.

## Поставщики

Поддерживается структура:

`Магазин → Камень → Номенклатурная группа → Поставщик`

Пустой поставщик и служебные варианты `Сеть`, `Unknown`, `Не указан` объединяются в `Other`.

## Локальный запуск

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
