# Аналитика 1.1.0

Настольное приложение Princess Jewelry для обработки Excel-выгрузок продаж по камням и жемчугу.

**Разработка: Vladimir Panasyan**

## Возможности 1.1.0

- одна общая выгрузка сразу по всем магазинам;
- поддержка старых отдельных выгрузок магазинов;
- возвраты полностью игнорируются — используются только колонки «Продано»;
- автоматическое определение периода;
- отдельный лист для каждого магазина и периода;
- несколько периодов одного магазина не склеиваются;
- сравнение периодов на листах `COMPARE`;
- диаграммы выручки, количества и структуры продаж;
- `SUMMARY`, листы магазинов и полный лист `RULES`;
- все неизвестные названия сохраняются в `OTHER STONES` собственной колонкой;
- автоматическая проверка обновлений через GitHub Releases.

## Как получить установщик без Python

1. Загрузите эти файлы в репозиторий GitHub.
2. Откройте вкладку **Actions**.
3. Выберите **Build Windows installer**.
4. Нажмите **Run workflow**.
5. После завершения скачайте artifact `Analitika_Setup_1.1.0`.

Шефам передается только файл `Analitika_Setup_1.1.0.exe`. Python им не нужен.

## Локальная сборка на Windows

1. Установите Python 3.12+ с опцией `Add Python to PATH`.
2. Установите Inno Setup 6.
3. Запустите `build_release.bat`.
4. Установщик появится в `installer_output`.

## Выпуск релиза

Создайте и отправьте тег:

```bash
git tag v1.1.0
git push origin v1.1.0
```

GitHub Actions автоматически соберет установщик и прикрепит его к Release.
