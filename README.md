# Analitika Web 0.1.3 TEST — Interactive Analytics

Тестовая веб-версия аналитики Princess Jewelry с интерактивным исследованием продаж.

## Новое в 0.1.3

- интерактивная цепочка `сегмент → камень → номенклатурная группа`;
- детальный разрез по серьгам, кольцам, подвескам, браслетам, ожерельям, жемчужным нитям и другим группам;
- количество, выручка, средняя стоимость, доля от количества и выручки магазина;
- при выборе конкретной номенклатурной группы — сравнение по всем магазинам;
- динамические диаграммы и таблицы;
- AI-выводы теперь учитывают лидирующую номенклатурную группу;
- исправлено обрезание больших сумм в карточках OUTLET.

## Локальный запуск

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Публикация в Streamlit Community Cloud

1. Скопируйте содержимое проекта в корень GitHub-репозитория.
2. Сделайте Commit + Push.
3. Streamlit Cloud автоматически обновит приложение.

Main file path: `streamlit_app.py`.

## Поставщики (0.1.3)

Загрузите вместе с основной выгрузкой файл с иерархией `Камень → Номенклатурная группа → Поставщик`.
В разделе **Поставщики** доступны общая сводка, рейтинг, категории и камни каждого поставщика.
Текущая supplier-выгрузка не содержит магазина, поэтому разрез по магазинам пока не рассчитывается.
