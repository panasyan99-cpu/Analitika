# Analitika Web 1.6.4

Внутренняя аналитическая платформа Princess Jewelry для четырех режимов:

- общий анализ продаж;
- сравнение двух периодов;
- склад Baserow;
- продажи и сетевые остатки Sonu.

## Что изменено в 1.6.4

- Единый фильтр **«Металл и пробы»** виден во всех четырёх режимах сразу после выбора режима и до курса валют.
- Выбор **Серебро / Золото и платина / Другое** сохраняется между вкладками.
- Общий анализ, сравнение и Sonu пересчитываются целиком по выбранным группам.
- Склад Baserow фильтруется по материалу позиции; поставки без построчной пробы остаются полными.
- После загрузки данных показываются конкретные распознанные пробы или материалы.
- Кнопка открытия боковой панели стала заметной золотой кнопкой с подписью **«МЕНЮ»**.

## Формат общего и сравнительного отчетов

Основной формат содержит иерархию:

`Магазин → Камень → Проба → Номенклатурная группа`.

Классификация проб:

- **Серебро** — B 925, Ag 925, 925 и распознанные варианты серебра;
- **Золото и платина** — любые AU и PT;
- **Другое** — Other 0, пустая проба и все значения, не относящиеся к серебру, золоту или платине.

## Sonu

- Остаток в файле трактуется как общий остаток сети и берется один раз на SKU.
- Продажи SKU суммируются по всем магазинам.
- Основная сводка: группа камней → вид камня → номенклатурная группа.
- Браслеты разделены на модели с затяжкой и без затяжки / полный круг.
- Доступно представление таблицами и карточками.

## Baserow

Складская аналитика выводится одной длинной страницей: обзор, сувениры, касты, позиции, требующие внимания, движение и поставки. Боковая навигация работает как переход к нужному разделу.

## Запуск

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

Публичная версия хранится в `version.json`. История изменений автоматически выводится из `CHANGELOG.md`.
