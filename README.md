# Analitika Web 1.1.0 RC1

Внутренняя интерактивная аналитическая платформа Princess Jewelry.

## Что исправлено

- один файл загружается один раз на всю сессию;
- все разделы строятся сразу на одной странице;
- боковое меню прокручивает к разделам;
- устранены дубли ключей Streamlit;
- поставщики выводятся автоматически;
- выводы расположены после каждого аналитического блока;
- выводы интерактивного блока зависят от выбранных фильтров;
- экспорт формируется только по запросу.

## Разделы

1. Сводка по сети
2. Магазины
3. Интерактивная аналитика
4. Поставщики
5. Экспорт

## Запуск локально

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
