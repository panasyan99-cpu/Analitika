## Beta-ограничение

Стартовое предлагаемое количество пока является прозрачной редактируемой подсказкой:

- продажи 0–2: 0;
- продажи 3–6 при рабочем остатке 0–3: 5;
- продажи 7+ при рабочем остатке 0–3: округление продаж вверх до кратного 5;
- положительный ТВП вычитается из подсказки;
- отрицательный ТВП не уменьшает заказ.

Эта формула вынесена в одну функцию `suggested_order_quantity()` и предназначена для уточнения после первой реальной проверки заказа.
