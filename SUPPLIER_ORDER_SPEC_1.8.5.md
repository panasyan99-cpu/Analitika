# Заказ поставщику — дополнение к ТЗ 1.8.5

## Входной файл

- Имя Excel-файла свободное и не участвует в определении формата.
- Структура определяется по заголовкам листа.
- При наличии колонки `NTR2` используется фактическое значение.
- Расчётный NTR2 разрешён только для старых файлов, где колонки нет.
- Фактический NTR2 не заменяется даже при расхождении с «Всего»; расхождение показывается как ошибка данных.

## Green Stones

- `Green Stones` — только крупный блок навигации.
- Внутри блока сохраняются отдельные названия: Emerald, Created Emerald, Chrome Diopside, Green Agate и Peridot.
- Карточка изделия, этап размеров колец и итоговый Excel используют конкретный камень.
- `Green Agat` автоматически исправляется на `Green Agate`.
- Если источник содержит только групповое название, конкретный камень восстанавливается по однозначному коду артикула; неразрешённый случай помечается как «Не распознано (Green Stones)».
