# Спецификация «Заказ поставщику» 1.9.4

## 1. Два остатка

- Отображаемый **«Общий остаток»** = `Всего − единый магазин 20/20NDC/Princess Hang`.
- TT и 63 входят в общий остаток и выводятся отдельной детализацией.
- Внутренний остаток спроса может исключать 63 и не должен называться общим остатком.

## 2. Собственный ТВП

Если `TVP > 0` у конкретного SKU:

- автоматическая рекомендация на этот же SKU равна нулю;
- доступен только ручной дозаказ;
- правило относится к серьгам, кольцам и остальным группам.

## 3. Баланс по ТВП внутри настоящего комплекта

Перекрёстная рекомендация разрешена только для `OrderSet.is_ungrouped = False`.

### Серьги в пути → кольца

1. Рассчитать прогнозируемое количество серёг к приходу: общий остаток + положительный ТВП − спрос на 2 месяца.
2. Целевое количество колец: `ceil(серьги × 0,6)`.
3. Из целевого количества вычесть прогнозируемое количество колец.
4. Минимальная партия рекомендации — 3 кольца.

### Кольца в пути → серьги

- 3–4 прогнозируемых кольца соответствуют 5 серьгам;
- каждые следующие четыре кольца добавляют ещё одну партию из 5 серёг;
- минимальная партия рекомендации — 5 серёг.

## 4. Слабые и нулевые реальные комплекты

При продажах 1–2, TT 0–2 и общем остатке меньше 5:

- серьги — 5;
- кольца — 3;
- подвески — 2.

При нулевых продажах, TT = 0 и общем остатке ниже базовой партии применяется та же минимальная рекомендация.

Если общий остаток равен примерно пяти и изделие уже представлено в TT, автоматический заказ не формируется.

## 5. «Без комплекта»

Каждый SKU оценивается независимо. Запрещено:

- использовать ТВП другого SKU;
- строить соотношение серьги/кольца;
- писать в причине о наличии других изделий комплекта.

При TT = 0:

- нулевой остаток → 5 серёг / 3 кольца / 2 подвески;
- остаток не выше продаж → полная базовая партия;
- остаток немного выше продаж, но ниже базовой партии → компактное пополнение TT: 3 / 2 / 1–2.

## 6. Двухмесячный спрос

Для продаж от 3 шт. сохраняется расчёт:

- месячный темп = продажи за период / 3;
- спрос до прихода = округлённый вверх темп × 2 месяца;
- если внутреннего доступного остатка недостаточно, рекомендация содержит двухмесячный объём.

Рекомендация никогда не записывается в заказ без кнопки «Согласен с рекомендацией».
