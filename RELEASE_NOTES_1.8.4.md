# Analitika Web 1.8.4

## Заказ поставщику

Версия исправляет границы ручных комплектов и делает ТВП рабочим фильтром заказа. Русские заголовки комплектов считываются как самостоятельные секции. В сводке ТВП отображается реальное суммарное количество изделий в пути.

### Основные изменения

- текстовые заголовки комплектов;
- сумма положительного ТВП в штуках;
- переключатель `ТВП > 0` / `ТВП ≤ 0`;
- фильтрация отдельных изделий без потери названия и категории комплекта.
