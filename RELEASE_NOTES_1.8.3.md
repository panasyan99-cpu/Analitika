# Analitika Web 1.8.3 — Stone-first supplier sets

Версия исправляет смешивание разных камней внутри одного ручного Set#. В исходном отчёте один Set# может содержать одну визуальную модель сразу в Blue Sapphire, Ruby, London Topaz и других камнях. Раньше приложение сначала собирало весь Set#, определяло общую категорию и только затем относило его к одному основному камню.

Теперь порядок обработки правильный:

1. нормализация названия камня;
2. разделение исходного Set# по камню;
3. расчёт категории, ТВП, ошибок и остатков внутри полученной каменной части;
4. вывод только этой части в выбранном камне.

Например, `Set# 2622` может отдельно присутствовать в Blue Sapphire и Ruby. Сапфировая карточка содержит только сапфиры и не получает категорию или ошибку ТВП от рубиновых изделий.
