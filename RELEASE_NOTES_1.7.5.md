# Analitika Web 1.7.5

## Ручной разбор спорных браслетов Sonu

- В блоке классификации отображается кнопка «Разобрать спорные модели».
- Открывается пошаговый мастер с одной крупной фотографией на экран.
- Для модели доступны два решения: «С затяжкой / центральная композиция» и «Без затяжки / полный круг».
- Показываются SKU, вид камня, продажи за период, продажи USD, остаток сети и автоматический предварительный выбор.
- Можно переходить назад и вперёд; сохранить набор разрешено только после выбора по всем моделям.
- После сохранения статус меняется на «Разобрано вручную», а решение получает наивысший приоритет.
- Сохранённые решения можно изменить, скачать резервным JSON или импортировать обратно.
- В облаке JSON рекомендуется добавить в репозиторий, чтобы решения не исчезли после нового деплоя.
